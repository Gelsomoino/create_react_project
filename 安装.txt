1. 全局安装
    npm i -g create-react-app

2. 切换到想创建项目的目录，使用
    create-react-app 项目名称     如：create-react-app my-app

3. 进入项目文件
   cd my-app

4. 启动项目 
   npm start  或 yarn start 
  (在学习过程中 ，建议使用 yarn start）





生成项目后
.git文件 
.gitignore  文件    git的忽略文件


在下载react 项目后 没有 node_modules 文件 需要 执行 npm i  安装所有依赖
